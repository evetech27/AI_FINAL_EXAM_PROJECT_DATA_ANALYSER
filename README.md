# School Data Insights Project

## Overview
This project analyzes student performance metrics using Python. It visualizes academic data to help educators identify trends in different units.

## Core Tasks
- [ ] Initialize Python data structures (Lists/Dictionaries).
- [ ] Implement Matplotlib for bar charts and trend lines.
- [ ] Calculate class averages and pass/fail ratios.

## How to Run
1. Ensure `matplotlib` is installed: `pip install matplotlib`
2. Run `python analyser.py`
### **System Logic & Persistence**
- **Data Format**: JSON (JavaScript Object Notation) used for structured storage.
- **Risk Assessment**: Implemented a threshold-based logic to flag subjects with low engagement (<75%).
- **I/O Operations**: Added `save_data` and `load_data` functions to ensure data survives program restarts.
